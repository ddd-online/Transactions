# Transactions 功能说明

## 软件概述

Transactions 是一款桌面个人财务管理工具，帮助用户记录和管理日常收支。软件围绕"多账本 + 分类标签 + 关键事件"三大核心概念组织，支持数据分析和图表可视化。

### 整体功能一览

- **多账本管理**：可创建多个独立账本（如"日常开支""生意收支"），每个账本拥有独立的分类体系和交易记录
- **消费记录**：记录每笔收入、支出或转账，支持分类、标签、时间等维度
- **模板系统**：将常用消费模式保存为模板，快速录入
- **数据筛选与排序**：按交易类型、分类、标签、关键词等条件组合筛选，支持多级排序
- **数据分析**：预设图表 + 自定义图表，从时间和分类维度可视化收支趋势
- **关键事件**：用日历标记人生重要节点，可关联相关交易记录、附加图片
- **明暗主题**：支持浅色 / 深色主题切换
- **本地存储**：所有数据保存在本地文件夹，每个工作空间为独立的数据库文件

---

## 页面功能

软件包含五个主页面，通过左侧导航栏切换。

---

### 一、消费记录（首页）

消费记录是日常使用最频繁的页面，用于浏览、查找、新增和管理交易。

**顶部工具栏**

- 时间范围快捷筛选：提供本月、上月、近三月等预设，也支持自定义起止日期
- 账本切换：下拉选择当前操作的账本，切换后自动加载对应数据

**交易列表**

- 每条记录展示：交易时间、类型（收入/支出/转账）、分类、标签、特殊标记、文字描述、金额
- 收入金额显示为绿色，支出为红色，转账为蓝色，便于快速区分
- 每行可执行三种操作：编辑、删除（需确认）、关联到关键事件

**底部分页**

- 显示当前记录总数
- 支持切换每页显示数量（15 / 30 / 50 / 100 条）

**排序**

- 浮层弹窗设置排序规则
- 最多支持 4 级排序，如"先按时间降序，再按金额升序"
- 可选排序字段：时间、金额、分类、类型
- 每级可独立选择升序或降序
- 支持添加、删除、重置排序条件

**筛选**

- 浮层弹窗设置筛选规则
- 支持多条筛选条件（条件之间为"或"的关系）
- 每条条件可指定：
  - 交易类型（收入 / 支出 / 转账）
  - 分类
  - 标签（可多选）
  - 标签匹配方式：满足任意一个标签，或满足全部标签
  - 标签取反：排除包含指定标签的记录
  - 描述关键词（模糊匹配）
- 支持一键清除所有条件

**新增 / 编辑交易**

点击悬浮按钮或表格中的编辑按钮弹出表单，包含以下字段：

- 模板选择：可从已保存的模板快速填充全部字段
- 交易时间：日期选择器
- 交易类型：收入 / 支出 / 转账（按钮切换）
- 分类：下拉选择，选项会根据交易类型自动变化
- 标签：多选下拉，选项会根据所选分类自动变化
- 标记：是否标记为"离群值"（异常大额消费等）
- 描述：自由文本
- 金额：输入数字，校验规则为 ≥0 且最多两位小数

表单可保存为新模板，用于下次快速录入同类交易。

**关联关键事件**

可将某笔交易与关键事件页面中的某个日期关联起来。已关联的交易会显示关联标记，点击可查看或解除关联。

---

### 二、账本管理

用于创建和管理多个账本，每个账本独立记账。

**账本卡片**

- 以卡片网格展示所有账本
- 第一个账本卡片会更大、更突出
- 每张卡片包含：账本名称、文字描述、创建日期
- 卡片左侧有彩色装饰条，不同账本自动分配不同颜色
- 支持编辑（修改名称和描述）和删除（需确认）
- 如果没有账本，显示引导提示

**新建账本**

- 名称：必填，最多 50 字
- 描述：选填，最多 200 字

---

### 三、数据分析

以图表形式直观展示收支数据的趋势和构成。

**顶部工具栏**

- 时间范围和账本切换，与消费记录页共享

**图表列表（左侧）**

- 预设两个不可删除的图表：
  - 收支趋势图
  - 分类支出图
- 用户可创建自定义图表

**图表展示（右侧）**

- 以折线图、柱状图、饼图等形式呈现数据
- 支持按年或按月两种粒度查看
- 自定义图表可自由配置：
  - 图表名称
  - 时间粒度（年 / 月）
  - 数据曲线：可选择追踪的收支类型、分类、标签
- 切换时间范围或账本时，所有图表数据自动刷新
- 底部显示当前条件下的收入、支出、转账汇总金额

---

### 四、关键事件

以全年日历的形式记录人生中的重要节点（如旅行、考试、换工作等），可附加文字、图片和关联交易记录。

**年份日历**

- 以 4 列网格同时展示全年 12 个月
- 窗口缩小时自动调整为 2 列或 1 列
- 通过左右箭头或年份选择器切换年份
- 每个日期格子：
  - 有事件的日期以颜色填充（颜色可自选）
  - 当天日期以红点标记
  - 鼠标悬停显示事件标题

**事件详情弹窗**

点击任意日期打开，窗口占据屏幕的约三分之二，包含两个标签页：

**详情标签页**：
- 标题：最多 200 字；保存时若未填写标题，自动提取正文第一行
- 颜色标记：20 种预设颜色可选，用于在日历中标识该事件
- 正文：最多 5000 字，自由记录
- 图片附件：
  - 最多支持多张图片，网格排列
  - 可点击选择文件上传，也可直接粘贴剪贴板中的图片
  - 图片可预览放大和删除

**关联交易标签页**：
- 列出与该日期关联的所有交易记录
- 顶部统计栏汇总关联交易的收入、支出、转账总额
- 可在此直接解除某笔交易的关联
- 标签页标题显示关联数量

**编辑与删除**

- 已有事件进入编辑模式，可修改全部内容，也可删除整个事件
- 新建事件仅显示保存按钮

---

### 五、应用设置

设置页面左侧为导航菜单，点击切换右侧内容。

**分类与标签**

- 顶部按交易类型（支出 / 收入 / 转账）切换
- 左侧为分类列表，右侧为对应分类下的标签列表
- 每个分类和标签都显示名称，分类额外显示使用该分类的记录数量
- 支持添加、删除、上下调整排序
- 操作按钮在鼠标悬停时显示

**工作空间**

- 显示当前工作空间所在的文件夹路径
- 提供"切换工作空间"按钮，选择另一个文件夹作为数据存储位置
- 切换后自动加载新工作空间的数据

**消费模板**

- 以表格列出当前账本下所有已保存的模板
- 每行展示：模板名称、交易类型、分类、标签、标记、描述
- 支持上下调整模板顺序、删除模板
- 模板也可在消费记录页的新增/编辑表单中保存和使用

**关于**

- 显示应用 Logo、名称和版本号
- 显示软件构建时间
- 显示版权信息
